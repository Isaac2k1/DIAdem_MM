MIDAS_ASC_DataPlugin.vbs

This is a data plugin for DIAdem which is suitable to import ASC-Files, created by MIDAS viewer. 

It was sucessfully tested with KEMA data, which have been exported using MIDAS viewer, version 1.2.0.4. The data was from KEMA tests in 1999, October 13th.
